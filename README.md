## Features

A huge content extension mod!

-   `180+`  Blocks

-   Custom Schematics

-   A trading system using gold as currency

-   Distribute items across the map with <br>
    little effort using the outpost block.

-   Lots of units

-   `2`  Larger Core Blocks

    *+ Fitting Drones*

-   `7`  Ammo Types
