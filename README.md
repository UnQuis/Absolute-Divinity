# Absolute Divinity

Content extension mod for [Mindustry](https://github.com/Anuken/Mindustry).

## Factions

- **Void** — white/silver energy weapons. 10 turrets.
- **Abyss** — purple/violet explosive and laser weapons. 10 turrets.
- **Order** — pure white precision weapons. 10 turrets.
- **Time** — not yet implemented.
- **Chaos** — not yet implemented.
- **Info** — not yet implemented.

## Content

- 60+ turrets across 6 factions
- Custom items, liquids, status effects
- Custom units and bosses
- Custom planets (Elysium, Verdict, Thanatos, Nyx)
- Custom weather types
- Custom bullet types (ChainBulletType, MassiveLaserBulletType)
- Custom planet generation and meshes

## Building

```bash
./gradlew build
```

Output: `build/libs/AbsoluteDivinity.jar`

Requires Mindustry v154.
