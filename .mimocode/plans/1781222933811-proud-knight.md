# Absolute Divinity — Обзор кодовой базы

## Общая информация
- **Название мода**: Absolute Divinity
- **Платформа**: Mindustry v154, Java 8 (DEX через D8), Gradle
- **Точка входа**: `ADMod.java` — наследует `mindustry.mod.Mod`
- **Всего Java файлов**: 112
- **Сборка**: `./gradlew build` — **BUILD SUCCESSFUL** (есть D8 desugaring warnings, но не ошибки)
- **Версия**: 0.0.1

---

## Архитектура

### Структура пакетов (`src/absolutedivinity/`)

| Пакет | Назначение |
|-------|-----------|
| `content/` | Все ресурсы мода: предметы, жидкости, статусы, оружие, планеты, погода, цвета |
| `content/blocks/` | Блоки: ядра, распределение, эффекты, турели (по фракциям) |
| `content/turrets/` | Лоадеры турелей: ADVoidTurrets, ADAbyssTurrets, ADChaosTurrets, ADTimeTurrets, ADInfoTurrets, ADOrderTurrets |
| `content/units/` | Юниты по тирам: t1 (8), t2 (1), t3 (1), t4 (1), idk/боссы (6), ракеты (8) |
| `content/entities/bullet/` | Кастомные типы снарядов: `ChainBulletType`, `MassiveLaserBulletType` |
| `core/` | Игровая логика, настройки, чат-обработчик |
| `core/map/` | Генерация планет, меш-рендеринг (Dyson ring, Dyson sphere, cylinder) |

## Ключевые особенности

### Игровая логика
- **Масштабирование HP врагов**: `ADGameLogic` — при спавне юнита Crux умножает HP на множитель из настроек (1-50x)
- **Настройки**: `ADSettings` — слайдер "ad-enemy-hp-mult", два чекбокса (пока не используются)

### Цветовая палитра
`ADColor`: ichor, divineBlood, deepRed, crimsonRed, richBlue, energyBlue

### Эффекты
`ADEffects` — 37 кастомных эффектов для Void и Order фракций (shoot, hit, charge, blast, trail и т.д.)

## Замеченные проблемы / особенности

1. **Заглушки**: Abyss/Chaos/Time/Info турели (40 штук) — полностью placeholder, без ammo/type/requires
2. **TechTree**: минимальная реализация (только 4 ноды: coreShard → ferrite → reinforced → matrix + ferroamalgam)
3. **ADChatHandler**: полностью закомментирован
4. **assets/ директория**: отсутствует (build.gradle ссылается на `from("assets/")`)
5. **D8 warnings**: desugaring warnings при сборке (некоторые типы не найдены — нормально для compileOnly)
6. **Funny Metal / Funny Effect**: мемный контент с Float.MAX_VALUE значениями
7. **Culiseta (T4)**: имеет только rotor-оружие, несмотря на описание с "fragging bombs, targetable missiles, railguns"
8. **ADCores**: объявлен `ascensionCore`, но не инициализирован
9. **reactiveRod**: объявлен в ADItems, но не определён в load()
10. **EradicatorPrime**: 8-ногий босс с MassiveLaserBulletType, FlakBulletType, LaserBulletType, ShieldArcAbility — самая продвинутая единица
