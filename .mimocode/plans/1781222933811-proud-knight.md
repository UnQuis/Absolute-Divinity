# Plan: Complex Turret Effects with Faction Colors

## Faction Color Scheme
- **Chaos**: red (#ff6b6b) + black (#1a1a1a)
- **Order**: white (#ffffff) + white (#f0f0f0)  
- **Void**: gray (#d0d4e8) + white (#ffffff)
- **Info**: green (#77df00) + white (#ffffff)
- **Abyss**: purple (#9000a4) + dark purple (#5a0070)
- **Time**: yellow (#f0e060) + white (#ffffff)

## Implementation Approach

### Phase 1: Create ADColors faction palette
Add faction-specific color constants to ADColor.java.

### Phase 2: Create ADTurretEffects parameterized factory
Create a new class with factory methods for each effect type:
- `shootSmall(color)`, `shootMedium(color)`, `shootBig(color)`
- `hitSmall(color)`, `hitMedium(color)`, `hitBig(color)`
- `chargeSmall(color)`, `chargeBig(color)`
- `trailEffect(color)`
- `blastEffect(color, size)`

Each uses:
- `Rand.setSeed(e.id)` for deterministic randomness
- `Drawf.tri()` for triangle patterns
- `Lines.circle()` for rings
- `Angles.randLenVectors()` for particles
- `Drawf.light()` for illumination
- Proper `Draw.z()` layering

### Phase 3: Update turrets per faction
- **Abyss**: Purple particle effects, corrosive theme
- **Chaos**: Red/black explosive effects, fire theme
- **Time**: Yellow/white temporal effects, freeze theme
- **Info**: Green/white precision effects, scanning theme

### Phase 4: Update ADEffects with faction-specific effects
Add new effects to ADEffects.java for each faction.

## Files to modify
1. `src/absolutedivinity/content/ADColor.java` - Add faction colors
2. `src/absolutedivinity/content/ADEffects.java` - Add faction effects
3. All 40 turret files - Update with proper effects
